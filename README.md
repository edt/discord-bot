# discord-bot

## Commandline Usage

The following arguments are available:

#### -h, --help

  Print arguments, etc.
  
#### -v[vvvv]

Set log level. Maximum is 5.

#### -c, --config <config.json>

Manually set the config that shall be used.

#### --reset

Purge cache before starting.

#### --data <directory>

Local directory for permanent local data.
